#  3D Developer Portfolio

This is my personal **3D Portfolio Website** built using modern web technologies.
It showcases my projects, skills, and experience in web development.

 **Live Website:**
https://codexshami.github.io/

---

## Built With

* React.js
* Vite
* Three.js
* Tailwind CSS
* Framer Motion
* EmailJS

---

##  Features

* Interactive **3D animations**
* Modern responsive UI
* Project showcase section
* Contact form with email integration
* Smooth animations and transitions

---

##  Project Setup

Clone the repository:

```bash
git clone https://github.com/codexshami/codexshami.github.io.git
```

Go to project directory:

```bash
cd codexshami.github.io
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

---

##  Deployment

This project is deployed using **GitHub Pages**.

To deploy changes:

```bash
npm run deploy
```
